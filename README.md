# Alfred-Timestamp

一个简单易用的 Alfred 时间戳转换工具，支持时间戳和标准时间格式之间的相互转换。

## 功能特点

- 显示当前时间和对应的时间戳
- 时间戳转换为标准时间格式
- 标准时间格式转换为时间戳
- 支持从剪贴板读取时间戳并转换

## 使用方法

1. 显示当前时间和时间戳：
```
tt now
```

2. 时间戳转标准时间：
```
tt 1488888888
```

3. 标准时间转时间戳：
```
tt 2017-03-07 20:14:48
```

4. 转换剪贴板中的时间戳：
```
tt
```

## 安装方法

1. 下载 `Timestamp.alfredworkflow` 文件
2. 双击安装到 Alfred
